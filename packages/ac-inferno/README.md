## Install

**npm**

```
npm install ac-inferno
```

**yarn**

```
yarn add ac-inferno
```

The [UMD](https://github.com/umdjs/umd) build is also available on [unpkg](https://unpkg.com):

```html
<script src="https://unpkg.com/ac-inferno@1.0.3/dist/ac-inferno.min.js"></script>
```
